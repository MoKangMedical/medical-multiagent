# Medical MultiAgent

多智能体医学研究平台 — 多个AI Agent协同完成医学研究任务

## 一句话定义

Medical MultiAgent 不卖AI工具，卖多学科协作的研究结果。每个Agent负责一个专业角色，协同产出完整研究方案。

## 核心Agent

- 文献Agent: 自动检索+综述
- 方法学Agent: 研究设计+统计方案
- 数据Agent: 数据清洗+分析
- 写作Agent: 论文初稿生成
- 审稿Agent: 模拟同行评审

## 快速开始

    git clone https://github.com/MoKangMedical/medical-multiagent.git
    cd medical-multiagent
    pip install -r requirements.txt
    python src/orchestrator.py
